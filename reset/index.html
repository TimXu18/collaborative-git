Reset

Workspace

  • Line 1
  • Line 2

Steps

  1. A create branch "reset-A" based on "master"
  2. B Create another branch "reset-B" based on "master"
  3. A modify line 1 and commit
  4. B modify line 1 and commit
  5. A modify line 2 but don't commit
  6. B modify line 2 but don't commit
  7. A run these commands 
    git merge reset-B
    git reset --merge
  8. B run these commands 
    git merge rebase-A
    git reset --hard

Summary

What's the difference? B will lose all of his changes. A will just undo the conflicted merge
