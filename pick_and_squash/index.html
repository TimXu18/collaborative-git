Cherry Pick

Workspace

  • Line 1
  • Line 2
  • Line 3

Steps

  1. A create branch "cherry-A" based on "master"
  2. A modify line 1 and commit
  3. B modify line 2 and commit
  4. A modify line 3 and commit
    5. git log
  5. A run the following commands 
    git rebase -i HEAD~3
  6. In the editor, change the last 2 "pick" to "squash". Save and exit 
    pick 
    squash 
    squash
  7. There will be a summary of the commits so far, this can be replaced as a new commit message. Save and exit
  8. Check the git log 
    git log

Summary

What's the difference? The second git lock will squash all three commits into one
